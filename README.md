# MooseFS-High-Availability-Cluster-Keepalived
Ce projet fournit une documentation complète pour déployer un système de stockage distribué MooseFS en haute disponibilité (HA) utilisant Keepalived pour la gestion automatique du failover. Solution idéale pour des environnements de production nécessitant une tolérance aux pannes et une disponibilité maximale des données.
